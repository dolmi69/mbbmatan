Алгебраические правила и формулы

Порядок действий

Сначала скобки, затем умножение и деление, потом сложение и вычитание.

Квадрат суммы

(a + b)² = a² + 2ab + b²

Квадрат разности

(a - b)² = a² - 2ab + b²

Разность квадратов

a² - b² = (a - b)(a + b)

Теорема Виета

Для квадратного уравнения ax² + bx + c = 0 сумма корней x₁ + x₂ = -b/a, произведение x₁·x₂ = c/a.

Дискриминант

D = b² - 4ac. Если D > 0 — два корня, D = 0 — один корень, D < 0 — корней нет.
