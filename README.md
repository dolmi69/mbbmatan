Проект "Mbb-duo" представляет собой веб-платформу, разработанную для интерактивного изучения математики и физики. Основная цель — предоставить пользователям доступ к учебным материалам (формулам, теории), инструментам для тестирования знаний и персонализации контента. Платформа построена на базе Django 5.2, что обеспечило быстрое создание функционала за счет встроенных модулей: ORM для работы с базой данных SQLite, системы аутентификации и шаблонизации.
В разработке использованы стандартные библиотеки Python, включая random для выбора случайных вопросов и json для хранения вариантов ответов. Frontend реализован на HTML/CSS с адаптивным дизайном, а система шаблонов Django позволила создать единую структуру страниц через наследование от base.html. Для безопасности применены CSRF-токены, валидация паролей и декораторы доступа (@login_required).
Ключевой функционал включает тестирование знаний с рандомным выбором 5 вопросов из 15, сохранение прогресса в сессиях и отображение результатов через прогресс-бар. Пользователи могут создавать заметки (CRUD-операции через Django Generic Views) и добавлять формулы в избранное. Администрирование контента осуществляется через панель Django Admin, где можно управлять вопросами, теорией и пользователями.
Архитектура проекта следует паттерну MVT (Model-View-Template). Модели (Formulas, Note, FormulaQuestion) описывают структуру данных, представления обрабатывают логику (например, проверку ответов в тестах), а шаблоны обеспечивают отображение. Маршрутизация настроена с использованием именованных URL и пространств имен для избежания конфликтов.
В процессе разработки решены типичные проблемы: бесконечные редиректы при тестировании (исправлены через флаги сессии quiz_started и quiz_completed), рандомизация вопросов (реализована через random.sample() и хранение ID в сессии), валидация ответов (сравнение user_answer с correct_answer).
Проект организован в структуру с виртуальным окружением (.venv) для изоляции зависимостей. Каталоги включают migrations для версий БД, static/css для стилей и templates для HTML-шаблонов. Кастомные фильтры (multiply, div) для расчета прогресса вынесены в templatetags/custom_filters.py.
Платформа готова к масштабированию: возможен переход на PostgreSQL, добавление разделов по новым дисциплинам (химия, информатика) и интеграция LaTeX для отображения формул. В перспективе — внедрение AJAX для динамической загрузки контента и системы достижений для повышения вовлеченности пользователей.
Проект демонстрирует эффективное использование Django для образовательных ресурсов, сочетая простоту разработки с гибкостью архитектуры.
